# 🎮 سامانه هوشمند تست و کالیبراسیون کنترلر (Controller Test & Calibration System)

یک وب‌اپلیکیشن مدرن، یکپارچه و فوق‌العاده دقیق برای تست، عیب‌یابی عمیق و کالیبراسیون پتانسیومترها و سنسورهای اثر هال انواع دسته‌های بازی (گیم‌پد). این سامانه با خروجی واقعی سخت‌افزار همگام‌سازی شده و به صورت اختصاصی برای پلتفرم **[Fix.Peyman24x.ir](https://Fix.Peyman24x.ir)** توسعه یافته است.

---

## ✨ ویژگی‌های کلیدی و اصلاحات نسخه جدید (v2.0)

* **🔄 اصلاح هندسی چیدمان آنالوگ‌ها (RTL Native):** باگ جابجایی بصری آنالوگ‌ها کاملاً برطرف شد. اکنون بردار آنالوگ راست دقیقاً در سمت راست صفحه و بردار آنالوگ چپ در سمت چپ تصویر متناسب با هندسه استاندارد رندر می‌شوند.
* **🎯 رفع باگ چرخش و پیمایش زوایا (گام ۳ جادوگر):** مجهز به ماتریس هوشمند جهت‌شناسی جغرافیایی. سیستم در مرحله سوم جادوگر، هر ۴ جهت اصلی (شمال، شرق، جنوب، غرب) را برای هر دو آنالوگ (مجموعاً ۸ گره فیزیکی) به صورت مجزا پایش می‌کند. تا زمانی که کاربر آنالوگ‌ها را ۳۶۰ درجه کامل نچرخاند و تمام دات‌های وضعیت سبز (`done`) نشوند، دکمه عبور به فاز ذخیره‌سازی فعال نخواهد شد.
* **🔋 پایش هوشمند باطری مابین پروتکل‌ها:** حل مشکل عدم نمایش باطری. در حالت استاندارد اگر مرورگر اطلاعات باطری را مسدود کند، داک هوشمند فعال شده و از کاربر درخواست سوئیچ به پروتکل WebHID را می‌کند. در حالت WebHID، پکت‌های خام فریمور مستقیماً پارس شده و ولتاژ پایدار به همراه وضعیت تغذیه نمایش داده می‌شود.
* **🔌 سوئیچینگ ریل‌تایم مابین APIها:** امکان جابجایی آنی مابین پروتکل استاندارد وب (`Standard Gamepad API`) و پروتکل بومی سطح پایین سخت‌افزار (`Low-Level WebHID API`) همراه با قطع و وصل هوشمند لوپ‌های پردازشی.
* **📊 محاسبه خطای هندسی دیستورشن (Circular Error):** محاسبه آنی و ریاضیاتی فرمول فیثاغورث بر روی مختصات خطی آنالوگ‌ها برای سنجش میزان دد-زون (Dead-zone) و خطای محیطی بیضی‌گون شاسی‌ها.
* **🎨 رابط کاربری مدرن شیشه‌ای (Glassmorphism):** دیزاین تاریک کاملاً واکنش‌گرا (Responsive) با تایپوگرافی خوانای فارسی و استایل‌های بهینه‌سازی شده بدون نیاز به فریم‌ورک‌های سنگین خارجی.

## 📂 ساختار فایل‌های پروژه

این پروژه بر پایه معماری فرانت‌اند تمیز و تفکیک شده (Vanilla Architecture) بدون وابستگی به پکیج‌های خارجی پیاده‌سازی شده است:

```text
├── index.html      # ساختار اصلی فرانت‌اند، رابط کاربری شیشه‌ای و کامپوننت‌های گرافیکی
├── app.js          # هسته هوشمند پردازشی، ماتریس جهت‌شناسی جادوگر و منطق محاسباتی APIها
└── README.md       # مستندات و راهنمای کامل پروژه (نسخه به‌روزرسانی شده)
🚀 راهنمای نصب و اجرا
پروژه به صورت ۱۰۰٪ کلاینت‌ساید (Client-Side) توسعه یافته و نیاز به هیچ‌گونه نصب پکیج یا سرور Node.js ندارد.

۱. ابتدا مخزن را کلون کنید یا فایل‌ها را دانلود نمایید:

۲. پوشه پروژه را باز کرده و فایل index.html را مستقیماً در مرورگر خود اجرا کنید.

💡 نکته فنی: به دلیل استفاده از قابلیت‌های نوین ابزار سخت‌افزاری وب مانند WebHID و Gamepad.battery، توصیه می‌شود پروژه را ترجیحاً بر روی مرورگرهای مبتنی بر کرومیوم مانند Google Chrome ،Microsoft Edge یا Opera اجرا نمایید.

🕹️ نحوه کار با جادوگر کالیبراسیون (Step-by-Step Wizard)
مرحله ۱ (اتصال پورت): کنترلر را متصل کرده و یکی از دکمه‌های آن را فشار دهید تا اتصال امن تایید شده و دکمه رفتن به گام بعدی باز شود.

مرحله ۲ (تراز مرکز): آنالوگ‌ها را کاملاً رها کنید. سیستم مقادیر آفست پتانسیومترها را بررسی می‌کند؛ در صورت قرارگیری در محدوده صفر مطلق (زیر ۰.۰۵)، تاییدیه صادر می‌شود.

مرحله ۳ (پیمایش زوایا): هر دو آنالوگ را یک دور کامل بچرخانید. با حرکت استیک‌ها به زوایای منتهی‌الیه شمال، شرق، جنوب و غرب، دات‌های راهنمای گرافیکی روی صفحه سبز می‌شوند. پس از سبز شدن تمام ۸ گره، اجازه ورود به فاز نهایی داده می‌شود.

مرحله ۴ (ذخیره‌سازی): در این مرحله تست‌ها با موفقیت پاس شده و رجیسترهای تصحیح خطا آماده رایت نهایی روی سیستم کلاینت فیکس هستند.

🛠️ تکنولوژی‌های مورد استفاده
HTML5 (ساختار معنایی و بهینه)

CSS3 Variables & Animations (رابط کاربری مدرن بدون وابستگی)

Vanilla JavaScript (ES6+) (هسته پردازش سیگنال فرکانس بالا)

HTML5 Canvas API (رندر پلتفرم برداری بردهای آنالوگ)

Web Gamepad API & WebHID API (ارتباط مستقیم و سطح پایین با فریمور سخت‌افزار)
